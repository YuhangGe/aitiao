# 爱跳跳（aitiao）

> ℹ️ **大模型驱动的自动跳广告工具**

> ⚠️ **声明：本项目提供安卓无障碍能力搭配视觉大模型能力的示例代码，不提供 APK，仅供开发者学习源代码。任何个人或组织克隆代码并构建 APK 后不得在公开渠道分发，一切违反法律犯规的行为自行承担后果，与本仓库无关。**

> 📢 **本项目代码完全使用 Claude Code + DeepSeek 生成，App 图标使用豆包生成。**


## 为什么有这个项目？

想用安卓上自动跳广告的工具，但“无障碍”权限实在是太敏感了，不放心，就自己写一个放心的工具。

## 这个项目有什么特色？

不走传统跳广告工具的路线，直接截图后让识图大模型判定下是不是广告页，如果是的话跳过按钮大概在哪个位置。然后点击对应位置。因此代码整体上非常简单，要审查有没有留后门或其它安全隐患也很容易。

大模型判定会有不小的延迟。所以会根据页面指纹进行缓存，后续检测到应用打开后，如果页面缓存找到了，直接通过点击缓存的跳过按钮位置来跳过广告，不需要走大模型判定。

如果应用有更新，换了跳过广告按钮位置，则需要清除下缓存，下次重新走大模型判定即可。

Release 包仅不到 3M 大小。

## 怎么使用这个项目？

1. 阅读项目所有代码，或者使用各种 agent + llm 来审计下代码，确保没有任何安全隐患自己可以放心使用。
2. 安装并配置好 android studio 开发环境后，直接构建项目并打包出 apk，安装到自己的设备上。
3. 在配置页把兼容 OpenAI 的大模型参数配置好。比如腾讯云混元、豆包，未来支持图片理解的 DeepSeek 等都行。
4. 整个过程如果遇到报签名文件找不到一类的问题，直接让 agent 帮解决就行。
